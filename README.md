# Learning Management System (LMS) 📚

A full-fledged **Learning Management System (LMS)** where **tutors can publish courses**, and **students can purchase and enroll** in them. The platform is **fully responsive** and supports **dark mode** for an enhanced user experience.

---

## 🌟 Features

- 📌 **Course Creation & Management** – Tutors can create, edit, and manage courses.  
- 🎓 **Student Enrollment & Purchase** – Students can purchase and access courses.  
- 🎨 **Beautiful UI with ShadCN** – Modern and sleek UI using **ShadCN UI components**.  
- 🌙 **Dark Mode Support** – Seamless light/dark mode switching.  
- 📱 **Fully Responsive** – Optimized for **mobile, tablet, and desktop**.  
- 🔐 **Authentication & Security** – Secure login and access control.  

---

## 🛠️ Technologies Used

This project is built using the **MERN stack**:

- **MongoDB** – NoSQL Database  
- **Express.js** – Backend Framework  
- **React.js** – Frontend Library  
- **Node.js** – Backend Runtime  
- **ShadCN UI** – Modern UI Component Library  
- **Redux Toolkit** – State Management  
- **Tailwind CSS** – Utility-First Styling  
- **Cloudinary** – Image & Video Hosting
